# Auto-Sync
- 每天4点自动同步上游Master分支
- ```Settings/secrets/SSH_KEY```中添加自己的SSH密钥
- ```Settings/secrets/USER_NAME```中添加自己的用户名
- ```Settings/secrets/USER_EMAIL```中添加自己的邮箱

# 注意
- 将上游地址github.com后面的路径放入forks.txt文件
- fork仓库名称和上游保持一致
